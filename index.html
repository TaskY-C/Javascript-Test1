  • == Equal Values

  • != Not Equal
  • !== Not Equal Value and Type
  • > Greater than
  • < Less than
  • >= Greater than or Equal
  • <= Less than or Equal

  • False && False ---False
  • True || True ---True
  • True || False ---True
  • False || True ---True
